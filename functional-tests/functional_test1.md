# Test Case ID: TC-Order-001

# Title:
    Verify ability to make an order

# Module:
    UI/Orders

# preconditions:
    --> this test must be carried out on the web application
    --> the user must be logged in

# Test Steps:
    1. click on the Orders cart on the left menu
    2. click on the create orders button on the top right corner
    3. press the add button on any meal to add it to cart.
    4. choose any compliment for the meal
    5. click on the create order button on the very bottom right corner.
    6. take note of the order id
    6. click on the orders menu and search for a pending order with the id you noted ealier.
    7. click on the made order that we created to verify the order was made.

# Expected Result:
    A page with the order we made should be shown. 
    The page should indicate all details about the order we made.

# Actual Result:
    The page does show the exact order made and its details.

# status:
    Test Passed

    N/a
