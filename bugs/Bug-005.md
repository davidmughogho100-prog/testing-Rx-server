# Bug ID: Bug-005

# Title:
    A Suplier's name can be set to a string of any length which breaks the UI

# Module:
    UI/Supliers

# Severity:
    Medium

# Priority:
    Medium

# Environment:
    android

# steps to reproduce
    1. login to the app
    2. click the Supliers menu
    3. click the add Suplier button at the bottom
    4. In the name field paste a very long string of text
    5. fill the rest of the fields 
    6. click the save button
    7. click the top back arrow button

# Actual Result:
    --> the user interface shows a very big suplier menu that stretch down

# Expected Result:
    --> the user interface should have a list of suppliers with their  names
    --> the suplier name input should not allow long strings to be entered
